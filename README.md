# QQBot 多插件项目

基于 NapCat + NoneBot2 的 QQ 机器人，支持多个插件。

![alt text](help/jmcomic/image.png)

## 插件列表

| 插件 | 说明 |
|------|------|
| **jmcomic** | 下载禁漫并生成 PDF 发送
| **pixiv** | Pixiv 搜索/下载/排行榜
